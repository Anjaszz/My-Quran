# PWA Icons

Untuk membuat PWA berfungsi dengan baik, Anda perlu membuat icon dalam berbagai ukuran.

## Cara Membuat Icons

Anda bisa menggunakan logo SVG yang ada di `src/assets/Logo.svg` dan mengkonversinya ke PNG dengan ukuran berikut:

### Ukuran yang Dibutuhkan:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## Tools Online untuk Generate Icons:

1. **PWA Asset Generator**: https://www.pwabuilder.com/imageGenerator
   - Upload logo SVG Anda
   - Download semua ukuran yang dibutuhkan

2. **RealFaviconGenerator**: https://realfavicongenerator.net/
   - Generate favicon dan PWA icons sekaligus

3. **Favicon.io**: https://favicon.io/favicon-converter/
   - Konversi SVG/PNG ke berbagai ukuran

## Cara Manual dengan Online Tool:

1. Buka https://www.iloveimg.com/resize-image/resize-svg
2. Upload file `src/assets/Logo.svg`
3. Resize ke ukuran yang dibutuhkan
4. Simpan dengan nama sesuai (contoh: icon-192x192.png)
5. Ulangi untuk semua ukuran

Setelah semua icon dibuat, letakkan di folder `public/icons/` ini.
